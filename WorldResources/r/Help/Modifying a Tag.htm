Modifying a Tag

By selecting to edit tags, you will be presented with a window that is much like the image below. All fields that are marked with * must be filled or selected, if you want to make a modification. If you do make a modification or delete a tag, changes will be present in the table and all other windows where tags are visible.
You can search for your tags by their ID. Every change in search will be registered, and you will see all tags of that ID as you type in the search box.
All other fields can be empty or not selected. If you make a wrong entry, you will be forced to change it and will be promped with a message.
