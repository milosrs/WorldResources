Using Main Window

When you enter Glowing Earth, you will see something similar to the picture below this text. You will be presented with a menu bar and a toolbar that are on top of the screen, a list of resources that you added on the right, and a world map in the center, with the resources icons and names below them, that you dragged into the map. If you change anything in a project, your title will get a star, like on the example below.

  • You can add an existing resource to the map by simply draging it onto the map. It will spawn right where you dropped it, if there are no resources that are too close to the desired location.

  • You can remove a resource from the map by simply right-clicking it. It will be removed from the map immediately.

  • Your resources must contain one type that you previously created, To create a type simpmly click the icon, or use Ctrl+Shift+T shortcut to enter the type creation window.

  • Your resource can contain none or more tags that you previously created. To create a tag click the icon, or use Ctrl+Shift+E shortcut to enter the tag creation window.

  • To create a new resource, click the icon, or use the Ctrl+Shift+R shortcut to enter the resource creation window.

  • You can create a new glowing earth project by clicking or using the Ctrl+N shortcut

  • You can save an activeglowing earth project by clicking or or using the Ctrl+S or Ctrl+Shift+S shortcut. If you save your project, and you had modified it, the star (*) will disappear from the title.

  • You can load an existing glowing earth project (.gemap file) by clicking or using the Ctrl+L shortcut

  • You can clear an active glowing earth project by clicking or using the Ctrl+N shortcut

  • You can modify resources, types and tags at any time by going to Edit in the Menu, and picking the wanted editor. Shortcuts are:

    • Alt+R for resources

    • Alt+T for types

    • Alt+E for Tags
